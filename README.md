# SmartClass SL Landing Page

This project is a landing page for **SmartClass SL**, an educational platform focused on making learning accessible for every student.

## Files

- `index.html` - Main landing page markup
- `main.css` - Page styling and responsive layout

## Features

- Hero section with educational branding and clear messaging
- Responsive navigation menu
- Feature cards for Study Notes, Video Tutorials, Past Papers, and Scholarships
- About section, testimonials, contact form, and footer
- Interactive hero background effect powered by CSS and JavaScript

## Usage

1. Open `index.html` in your browser, or use Live Server in VS Code.
2. Refresh the page after making edits.

## Notes

- The hero section currently uses a background image to present the education theme.
- Update `main.css` and `index.html` to customize content, colors, or layout.
